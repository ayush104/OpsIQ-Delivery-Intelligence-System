# 🚚 OpsIQ — AI-Powered Delivery Operations Intelligence System

**Author:** Ayush Saini  
**Domain:** Quick Commerce & Last-Mile Logistics  
**Tools:** Python | SQLite | Advanced SQL | Groq Llama3 API | Scikit-learn | Streamlit | Plotly  

---

## Problem Statement
India's quick commerce companies promise delivery in under 20 minutes. Every SLA breach causes refunds and customer churn. OpsIQ solves this by detecting failures, explaining why they happen, and telling operations what to do.

---

## Tech Stack
| Layer | Technology |
|---|---|
| Database | SQLite — 6-table relational schema |
| Analytics | Advanced SQL — CTEs, Window Functions, RANK(), LAG() |
| AI Engine | Groq Llama3 API |
| ML | Logistic Regression |
| Dashboard | Streamlit + Plotly |

---

## Key Results
- 50,000 delivery records across 6 zones
- Whitefield zone — 35% SLA breach rate
- ₹37.6L revenue at risk from breaches
- 16.9% return rate prediction accuracy
- AI auto-generates weekly playbook

---

## How to Run

Open the notebook file in Google Colab and run all cells in order.
The database and all tables are created automatically inside the notebook.
No separate setup required.
